# Roadbook · Schwarzwald & Vogesen · 14.–17. Oktober 2026

Interaktives Roadbook für die Tour durch Schwarzwald, Vogesen und Elsass.

## Inhalt

- `index.html` — Interaktive Web-Version (Leaflet-Karten, Wetter-Forecast, klickbare Hotels/Restaurants)
- `Roadbook_Schwarzwald_Vogesen_14-17_Oktober_2026.pdf` — Druckfertige PDF-Version

## Live-Version

Auf GitHub Pages aktivieren:

1. Repo erstellen, beide Dateien hochladen
2. Settings → Pages → Source: `main` / Root → Save
3. Nach ~1 Minute live unter `https://<username>.github.io/<repo>/`

## Features

- 🗺️ **Interaktive Karten** für Gesamtroute + jede Etappe (OpenStreetMap + OSRM-Routing)
- 🌤️ **Live-Wetter** für alle vier Tage (Open-Meteo API)
- 📞 **Klickbare Telefonnummern** und Google-Maps-Links für alle Hotels und Restaurants
- 📱 **Mobile-optimiert** — funktioniert genauso gut auf Handy wie Desktop
- 📄 **PDF-Download** direkt aus der Webseite

## Tech

Single-File HTML mit Leaflet.js (CDN). Keine Build-Tools, keine Dependencies, kein API-Key nötig.
